## Phase 1: Smart Frame Extraction

### Mục tiêu
Extract 300–400 frames nét nhất từ video trận đấu, trong đó **vùng torso (logo area) phải thật sự nét**, không chỉ toàn frame.

### 1.1 Thay đổi so với pipeline hiện tại

Pipeline hiện tại (`run_extraction.py` + `src/frame_extraction/`) đã rất tốt. Chỉ cần **3 thay đổi nhỏ**:

#### Thay đổi 1: Torso Sharpness (thay vì whole-player sharpness)

**File**: `src/frame_extraction/helpers.py`

**Hiện tại**: `compute_player_sharpness()` đo sharpness trên toàn bộ player bbox.
**Vấn đề**: Chân/tay có thể sharp nhưng torso (nơi có logo) bị blur do xoay người.
**Sửa**: Thêm hàm `compute_torso_sharpness()` — chỉ đo vùng 15%-65% height của player bbox.

```python
def compute_torso_sharpness(frame, detections, overlay_mask=None):
    """
    Compute sharpness CHỈ trên TORSO region (nơi logo nằm).
    Torso = 15%-65% height từ trên xuống của player bbox.
    """
    if not detections:
        return compute_sharpness(frame, overlay_mask)

    h, w = frame.shape[:2]
    scores = []

    for det in detections:
        x1, y1, x2, y2 = [int(v) for v in det["bbox"]]
        bh = y2 - y1

        # Torso region: 15%-65% of player height
        torso_y1 = max(0, y1 + int(bh * 0.15))
        torso_y2 = min(h, y1 + int(bh * 0.65))

        if torso_y2 <= torso_y1 or x2 <= x1:
            continue

        torso = frame[torso_y1:torso_y2, x1:x2]
        if torso.size == 0:
            continue

        # Overlay mask for torso region
        torso_mask = None
        if overlay_mask is not None:
            torso_mask = overlay_mask[torso_y1:torso_y2, x1:x2]
            if torso_mask.mean() < 0.4:
                continue

        scores.append(compute_sharpness(torso, torso_mask))

    return round(max(scores), 4) if scores else 0.0
```

**File**: `src/frame_extraction/selection.py`

```python
TIER_PRIORITY = {"gold": 0, "silver": 1, "bronze": 2}

def _pick_with_diversity(pool, n_want, min_gap_sec):
    if not pool:
        return []

    # Sort: gold first, then by score within each tier
    pool_sorted = sorted(pool, key=lambda x: (
        TIER_PRIORITY.get(x.get("sharpness_tier", "bronze"), 2),
        -x["score"]
    ))

    picked = []
    for candidate in pool_sorted:
        if len(picked) >= n_want:
            break
        ts = candidate["timestamp_sec"]
        if all(abs(ts - p["timestamp_sec"]) >= min_gap_sec for p in picked):
            picked.append(candidate)

    return picked
```

## Phase 1b: Temporal Annotation Propagation

### Mục tiêu
Từ 400 frames manual annotation → tạo thêm ~1,600 frames auto-annotated từ neighbor frames → total **~2,000 training frames**.

### 1b.1 Workflow

```
1. Annotate 400 frames trên Roboflow (MANUAL)
2. Export annotations (YOLO format)
3. Chạy propagation script:
   Với mỗi annotated frame tại thời điểm T:
     → Đọc frames T-3, T-2, T-1, T+1, T+2, T+3 từ video
     → Track mỗi bbox bằng template matching
     → Nếu match confidence > 0.6 → copy annotation với bbox mới
     → Gán sharpness tier cho neighbor frame
     → Save neighbor + adjusted annotation
4. Output: ~2,000 frames với YOLO annotations
```

### 1b.2 Implementation: `propagate_annotations.py`

```python
"""
Temporal Annotation Propagation.
Input:  annotated frames (YOLO format) + video file
Output: additional annotated frames from neighbors
"""

def propagate_single_frame(video_path, frame_num, annotations, radius=3):
    """
    Propagate annotations from frame_num to ±radius neighbors.

    Args:
        video_path: path to video
        frame_num: center frame (has manual annotations)
        annotations: list of [class_id, cx, cy, w, h] (YOLO format, normalized)
        radius: ±N frames to propagate to

    Returns:
        list of (neighbor_frame_num, neighbor_frame, adjusted_annotations)
    """
    cap = cv2.VideoCapture(str(video_path))
    total = int(cap.get(cv2.CAP_PROP_FRAME_COUNT))

    # Read center frame
    cap.set(cv2.CAP_PROP_POS_FRAMES, frame_num)
    ret, center_frame = cap.read()
    if not ret:
        cap.release()
        return []

    h, w = center_frame.shape[:2]
    center_gray = cv2.cvtColor(center_frame, cv2.COLOR_BGR2GRAY)

    results = []

    for offset in range(-radius, radius + 1):
        if offset == 0:
            continue
        nb_fn = frame_num + offset
        if nb_fn < 0 or nb_fn >= total:
            continue

        cap.set(cv2.CAP_PROP_POS_FRAMES, nb_fn)
        ret, nb_frame = cap.read()
        if not ret:
            continue

        nb_gray = cv2.cvtColor(nb_frame, cv2.COLOR_BGR2GRAY)

        # Track each annotation bbox
        adjusted = []
        for ann in annotations:
            cls_id, cx, cy, bw, bh = ann

            # Convert normalized YOLO → pixel coords
            px1 = int((cx - bw/2) * w)
            py1 = int((cy - bh/2) * h)
            pw = int(bw * w)
            ph = int(bh * h)

            # Extract template from center frame
            px1c = max(0, px1)
            py1c = max(0, py1)
            template = center_gray[py1c:py1c+ph, px1c:px1c+pw]
            if template.size == 0 or template.shape[0] < 10 or template.shape[1] < 10:
                continue

            # Expand search region (player moves ~15px/frame)
            expand = abs(offset) * 20
            sx1 = max(0, px1 - expand)
            sy1 = max(0, py1 - expand)
            sx2 = min(w, px1 + pw + expand)
            sy2 = min(h, py1 + ph + expand)

            search = nb_gray[sy1:sy2, sx1:sx2]
            if search.shape[0] < template.shape[0] or search.shape[1] < template.shape[1]:
                continue

            result = cv2.matchTemplate(search, template, cv2.TM_CCOEFF_NORMED)
            _, max_val, _, max_loc = cv2.minMaxLoc(result)

            if max_val > 0.5:  # Sufficient match confidence
                # New bbox position
                new_x1 = sx1 + max_loc[0]
                new_y1 = sy1 + max_loc[1]

                # Convert back to YOLO normalized
                new_cx = (new_x1 + pw/2) / w
                new_cy = (new_y1 + ph/2) / h

                adjusted.append([cls_id, new_cx, new_cy, bw, bh])

        if len(adjusted) >= len(annotations) * 0.5:  # At least 50% tracked
            results.append((nb_fn, nb_frame, adjusted))

    cap.release()
    return results
```

## Phase 2: Model Training (Crop-and-Detect)

### 2.1 Kiến trúc Detection

**KHÔNG** detect logo trên full frame 1920×1080.
**CÓ** detect person → crop player → detect logo trên crop 640×640.

```
Full frame (1920×1080)
    → YOLOv11l detect persons       ← pretrained, không cần train
    → Crop từng player (pad 20%)
    → Resize crop → 640×640
    → YOLOv11m detect logos         ← fine-tuned trên crops
```

**Tại sao crop-and-detect:**
- Logo 40px trên full frame → 150-250px trên crop = **4-6x magnification**
- mAP tăng ước tính: 35-45% (full frame) → 75-85% (crop)
- Tự loại bỏ LED board logos (nằm ngoài player bbox)

### 3.3 Stage 2: Team Classification

Simple color-histogram approach — không cần ML:

```python
def classify_team(player_crop_bgr, kit_type="home"):
    """
    Classify by dominant color in torso region.
    kit_type: "home" (black) or "away" (white)
    """
    h, w = player_crop_bgr.shape[:2]
    torso = player_crop_bgr[int(h*0.2):int(h*0.6), int(w*0.2):int(w*0.8)]
    hsv = cv2.cvtColor(torso, cv2.COLOR_BGR2HSV)

    if kit_type == "home":
        # Bradford Black: V < 60, S < 80
        black_mask = (hsv[:,:,2] < 60) & (hsv[:,:,1] < 80)
        return "bradford" if black_mask.mean() > 0.35 else "opponent"
    else:
        # Bradford White: V > 180, S < 50
        white_mask = (hsv[:,:,2] > 180) & (hsv[:,:,1] < 50)
        return "bradford" if white_mask.mean() > 0.35 else "opponent"
```

### 3.4 Stage 3: Selective Keyframe Logo Detection

**KHÔNG detect logo trên mọi frame** — chỉ trên keyframes (tiết kiệm compute):

```python
def select_keyframes(tracked_players, logo_tracker, fps=30):
    """
    Adaptive keyframe selection:
    - New tracks: detect every frame (establish binding quickly)
    - Unstable binding: every 5 frames
    - Stable binding: every 15 frames (just confirmation)
    """
    keyframes = set()

    for frame_num, players in tracked_players.items():
        for track_id in players:
            track = logo_tracker.get(track_id)

            if track is None or track.frames_seen < fps:
                keyframes.add(frame_num)  # New → every frame
            elif not track.has_stable_binding():
                if frame_num % 5 == 0:
                    keyframes.add(frame_num)  # Unstable → every 5
            else:
                if frame_num % 15 == 0:
                    keyframes.add(frame_num)  # Stable → every 15

    return keyframes

# Result: ~20% of frames processed instead of 100%
# Compute savings: ~5x
```

### 3.5 Stage 4: Logo-Player Binding

```python
class LogoTracker:
    """Accumulate logo detections per tracked player."""

    def __init__(self, min_binding_count=3, min_binding_confidence=0.6):
        self.tracks = {}
        self.min_count = min_binding_count
        self.min_conf = min_binding_confidence

    def update(self, track_id, logo_detections, frame_num):
        if track_id not in self.tracks:
            self.tracks[track_id] = {
                "frames_seen": 0,
                "logo_counts": defaultdict(int),
                "logo_confidences": defaultdict(list),
            }

        track = self.tracks[track_id]
        track["frames_seen"] += 1

        for det in logo_detections:
            if det.confidence >= self.min_conf:
                track["logo_counts"][det.class_name] += 1
                track["logo_confidences"][det.class_name].append(det.confidence)

    def get_bound_logos(self, track_id):
        """Return confirmed logos (detected >= min_count times)."""
        track = self.tracks.get(track_id)
        if not track:
            return []

        bound = []
        for logo_class, count in track["logo_counts"].items():
            if count >= self.min_count:
                avg_conf = np.mean(track["logo_confidences"][logo_class])
                bound.append({
                    "class": logo_class,
                    "confidence": avg_conf,
                    "count": count,
                    "rate": count / track["frames_seen"],
                })
        return bound

    def has_stable_binding(self, track_id):
        logos = self.get_bound_logos(track_id)
        return len(logos) >= 1
```

#### 2. Quality-Weighted Exposure

```python
def visibility_weight(player_bbox, frame_size, n_players_in_frame):
    """Quality Index for single exposure instance."""
    bw, bh = bbox_width(player_bbox), bbox_height(player_bbox)

    # Size: larger player → logo more visible
    size = (bw * bh) / (frame_w * frame_h)
    size_score = min(size * 50, 1.0)

    # Position: center of screen → more viewer attention
    cx, cy = bbox_center(player_bbox)
    dist = sqrt((cx/frame_w - 0.5)**2 + (cy/frame_h - 0.5)**2)
    position_score = max(0, 1.0 - dist * 1.5)

    # Clutter: fewer players → less attention competition
    clutter_score = 1.0 / (1 + 0.1 * n_players_in_frame)

    return size_score * position_score * clutter_score

# Weighted exposure calculation:
for event in exposure_events:
    weight = visibility_weight(event.bbox, frame_size, n_players)

    if event.source == "detected":
        weighted_exposure[event.logo] += weight * event.confidence / fps
    else:  # inferred
        weighted_exposure[event.logo] += weight * event.confidence * 0.7 / fps
```

#### 3. Continuous Exposure Segments

```python
def find_segments(logo, fps, min_duration=0.5, max_gap=0.2):
    """
    Group consecutive frames into exposure events.
    Only count segments >= 0.5s (brain recognition threshold).
    """
    frames = sorted(logo_frames[logo])
    segments = []
    current_start = frames[0]

    for i in range(1, len(frames)):
        gap = (frames[i] - frames[i-1]) / fps
        if gap > max_gap:
            duration = (frames[i-1] - current_start) / fps
            if duration >= min_duration:
                segments.append({
                    "start": current_start / fps,
                    "end": frames[i-1] / fps,
                    "duration": duration,
                })
            current_start = frames[i]

    return segments
```

## Phase 5: Edge Cases & Mitigations

### Replay Detection
- Replay segments → double-count exposure
- Solution: detect scene transitions + scoreboard overlay changes
- Flag replay frames, report separately

### Goalkeeper
- Different kit → different logo set
- Solution: team classifier handles "bradford_gk" as separate class

### LED Board Logos
- Crop-and-detect architecture **inherently solves this**
- Logo detection only runs on player crops → LED boards excluded

### Track ID Swap (Ruck/Maul)
- 6-8 players pile up → tracker swaps IDs
- Solution: during ruck events (detected by high bbox overlap), freeze logo binding updates

```
3. Target: mAP@50 > 0.75

```

### Phase 3 Validation: End-to-End Exposure Accuracy
```
1. Select 3 × 5-minute video segments (different game phases)
2. Manual ground truth:
   - Frame by frame: which Bradford players visible?
   - Which logos visible? How long?
3. Run pipeline on same segments
4. Compare:
   - Exposure time deviation: |predicted - actual| / actual < 15%
   - Logo ranking correlation: Spearman ρ > 0.90
   - Precision @ conf=0.5: > 0.90
```
